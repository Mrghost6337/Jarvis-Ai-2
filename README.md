## 📖 Overview

**Jarvis AI 2** is a next-generation voice-first AI assistant designed to redefine human-computer interaction. Unlike traditional chat-based bots, Jarvis functions as a proactive, continuous-listening assistant that springs to action only when called. 

It interprets natural human speech, engages in fluid conversations, executes complex system commands, and dynamically generates intelligent user interfaces to display results. Whether you need an update on global events, automated system management, or a functional website built on the spot, Jarvis is equipped to listen, process, and build in real time.

## ✨ Features

- 🎤 **Voice-first interaction:** Hands-free, voice-driven experience. No chat typing required.
- 🧠 **Real conversational AI:** Fluid, context-aware, and natural dialogues.
- 🌍 **Multi-language support:** Instantly detecting and switching between languages mid-conversation.
- ⚡ **Instant command execution:** Real-time processing with minimal latency.
- 💻 **Full system control:** Execute deep system scans, diagnostics, and tasks seamlessly.
- 🌐 **Smart web search with visual UI:** Generates beautiful, dynamic visual summaries of news, weather, and web data.
- 🧑‍💻 **App & website builder:** Can scaffold and write code instantly based on voice prompts.
- 🎨 **Liquid Glass UI:** A stunning, modern, and adaptive graphical interface focusing on aesthetics and usability.

## ⚙️ How It Works

1. **Wake Word Detection:** The system continuously listens in the background, consuming minimal resources, and activates only when it hears the word **"Jarvis"**.
2. **AI Processing:** The audio is streamed directly to advanced Language and Audio models to interpret intent and context almost instantly.
3. **Execution:** Jarvis queries the web, writes code, or manages tasks depending on the command.
4. **Voice Response:** Jarvis responds back using a highly natural human voice, explaining *what* happened and *why*, avoiding robotic lists.
5. **Dynamic GUI:** If visual information is required (like code, a task list, or news), a dynamic Liquid Glass interface opens seamlessly on the screen.

## 🚀 Installation Guide

Get Jarvis AI 2 running on your local machine with these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mrghost6337/Jarvis-Ai-2.git
   cd jarvis-ai-2
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file and add your required API keys (e.g., Gemini Flash AI):
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

4. **Run the App**
   ```bash
   npm run dev
   ```
